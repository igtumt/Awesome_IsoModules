
# 📦 Awesome IsoModules

A curated list of awesome WebAssembly (WASM) modules built for the **IsoBrowse** ecosystem. 
Want to add your module? Submit a Pull Request!

## 🛠️ Utilities & Tools

| Module Name | Description | Command (Copy/Paste) | Developer |
| :--- | :--- | :--- | :--- |
| **PDF-Merge** | Merges local PDF files securely in RAM. | `> /fetch github.com/user/pdf.wasm` | [@user](https://github.com/user) |
| **Log-Parser** | Finds CRITICAL errors in large server logs. | `> /fetch github.com/user/log.wasm` | [@devguy](https://github.com/devguy) |
| **Crypto-Ticker**| Live Bitcoin prices in the terminal. | `> /fetch github.com/user/btc.wasm` | [@cryptofan](https://github.com/cryptofan) |

---
*Disclaimer: These modules are created by the community. IsoBrowse runs them in a strict WASI P1 sandbox for your security.*
